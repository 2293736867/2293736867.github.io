VirtualBox CentOS8 调整分辨率

2020-08-19 VirtualBox ， CentOS

1 概述

VirtualBox安装完CentOS8后无法调节分辨率，需要安装额外的工具。

2 安装依赖包

首先确保虚拟机能正常连接网络，然后安装：kernel、kernel-core、kernel-modules、kernel-devel、gcc、make、perl、elfutils-libelf-devel：

sudo yum install kernel kernel-core kernel-modules kernel-devel \
gcc make perl elfutils-libelf-devel

检查kernel源码是否与内核版本是否一致：

ls /usr/src/kernels
uname -r
