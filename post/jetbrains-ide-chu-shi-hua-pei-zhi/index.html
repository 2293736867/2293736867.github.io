JetBrains IDE初始化配置

2020-08-24 JetBrains

1 初衷

这是给新下载的JetBrains IDE的一些好用的建议，包括插件，快捷键以及常用的设置，以IDEA为例。

只是建议，根据自己的需要选择修改即可。

2 常用设置

  • 外观字体调节：Use custom font
  • 设置背景：Background Image
  • 代理：HTTP Proxy
  • 关闭共享数据：Data sharing
  • 关闭自动更新：Updates
  • 修改Tab位置：Tab placement
  • 编辑器字体：Font
  • 主题：Color Scheme
  • 编码：File Encodings

3 快捷键

  • 空格补全：Choose Lookup Item Replace添加空格
  • 主菜单栏：Main Menu，Ctrl+Alt+Shift+C
  • 工具栏：Toolbar，Ctrl+Alt+Shift+B
  • 导航栏：Navigation Bar，Ctlr+Alt+Shift+X
  • 工具窗口栏：Tool Window bars，Ctlr+Alt+Shift+Z
  • 状态栏：Status Bar，Ctrl+Alt+Shift+V
  • 面包屑：Members in navigation bar，Ctrl+Alt+Shift+B
  • 重命名：Rename，F2
  • 新建：New，Ctrl+Shift+N
  • 显示数据库：Database，Alt+2
  • 显示插件：Plugins，Ctlr+Alt+Shift+A

4 插件

建议禁用不需要的插件，禁用后建议安装如下插件：

  • Atom Material Icons
  • Grep Console
  • IdeaVim
  • JRebel
  • Lombok
  • Power Mode II
  • Rainbow Brackets
  • HighlightBracketPair

5 VM Options

遵循一个简单的原则：给大内存就行了。

-Xms2g
-Xmx4g
-XX:ReservedCodeCacheSize=1g
