为Github的README.md生成目录的小工具

2020-09-26 Java

1 概述

因为Github的README.md文件[TOC]不生效，因此封装了一个别人已封装好的JAR包供大家使用。

2 使用方法

用Java做的，只需要JDK11以上的环境：

java -jar toc.jar 1.md  # 为1.md文件生成目录
java -jar toc.jar 1     # 为文件夹1内的.md生成目录

  • CODE.CHINA
